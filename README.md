
# Adidas Webstore Data Analysis (Power BI)

This project focuses on analyzing Adidas webstore product data using **Power BI**, **Python**, and **SQL**. The goal of the analysis is to identify sales trends, product characteristics, price distributions, and customer usage categories, as well as to present insights through an interactive and user-friendly dashboard.
